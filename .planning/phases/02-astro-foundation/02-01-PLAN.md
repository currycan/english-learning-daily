1. Create `website/src/pages/index.astro` (per D-03 — minimal lesson list, UI-SPEC Copywriting):
   ```astro
   ---
   import { getCollection } from 'astro:content';
   import Base from '../layouts/Base.astro';

   const entries = await getCollection('lessons');
   const lessons = entries.sort((a, b) => b.id.localeCompare(a.id));
   ---
   <Base title="English Learning Daily">
     <h1 class="text-xl font-semibold mb-4">Lessons</h1>
     {lessons.length === 0 ? (
       <div>
         <h2 class="text-xl font-semibold">No lessons yet</h2>
         <p class="mt-2">Lessons appear here once content is committed to the repository.</p>
       </div>
     ) : (
       <ul class="space-y-2">
         {lessons.map(lesson => (
           <li>
             <a href={`/lesson/${lesson.id}`} class="text-[#2563eb] hover:underline">
               {lesson.id}
             </a>
           </li>
         ))}
       </ul>
     )}
   </Base>
   ```
   CRITICAL per RESEARCH.md Pitfall 2: Always sort explicitly after getCollection — sort order is non-deterministic.
   CRITICAL per RESEARCH.md Pitfall 3: Link href uses `/lesson/${lesson.id}` — Astro auto-prefixes with `base` in .astro files.
   Heading "Lessons" and empty state copy per UI-SPEC Copywriting Contract.
   Accent color #2563eb per UI-SPEC Color table.
   List spacing `space-y-2` (8px) per UI-SPEC Layout Contract.

2. Create `website/src/pages/lesson/[id].astro` (per D-04 — stub pages rendering raw Markdown):
   ```astro
   ---
   import { getCollection, render } from 'astro:content';
   import Base from '../../layouts/Base.astro';

   export async function getStaticPaths() {
     const lessons = await getCollection('lessons');
     return lessons.map(lesson => ({
       params: { id: lesson.id },
       props: { lesson },
     }));
   }

   const { lesson } = Astro.props;
   const { Content } = await render(lesson);
   ---
   <Base title={lesson.id}>
     <h1 class="text-xl font-semibold mb-4">{lesson.id}</h1>
     <article class="prose">
       <Content />
     </article>
   </Base>
   ```
   Uses `render(entry)` free function (Astro 5+ pattern, NOT the deprecated `entry.render()` method).
   Stub title is the lesson ID (date string) per UI-SPEC Copywriting.
   Layout import path is `../../layouts/Base.astro` (two levels up from `pages/lesson/`).
