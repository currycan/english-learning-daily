**File header and imports:**
```python
"""RSS article fetcher — Phase 2.

Fetches one English article per day from configured RSS feeds.
Outputs Article Envelope JSON to stdout for Phase 3 consumption.

Usage:
    python -m scripts.feed_article
"""
import html as html_module
import json
import re
import sys
from html.parser import HTMLParser
from pathlib import Path

import feedparser

from scripts.content_utils import content_path, get_beijing_date
```

**_TextExtractor (private HTMLParser subclass, not exported):**
- Subclass HTMLParser
- Class attribute: BLOCK_TAGS = {"p", "br", "div", "h1", "h2", "h3", "h4", "li"}
- __init__: super().__init__(); self._parts: list[str] = []
- handle_data(self, data: str) -> None: self._parts.append(data)
- handle_starttag(self, tag: str, attrs) -> None: if tag.lower() in self.BLOCK_TAGS: self._parts.append("\n")
- get_text(self) -> str: return "".join(self._parts)

**clean_html(raw: str) -> str:**
- Instantiate _TextExtractor, call extractor.feed(raw), get text via extractor.get_text()
- Apply html_module.unescape(text)
- Collapse 3+ consecutive newlines: re.sub(r"\n{3,}", "\n\n", text)
- Return text.strip()
- Contract: no HTML tags in output; entities decoded; "AT&amp;T" → "AT&T"; "<p>Hello <strong>world</strong></p>" → "Hello world" (no tags)

**_extract_body(entry) -> str:**
- Priority chain: if entry.get("content"): return entry["content"][0]["value"]
- elif entry.get("summary"): return entry["summary"]
- else: return entry.get("description", "")
- Contract: entry with both content[0].value and summary → returns content[0].value

**_validate(envelope: dict) -> bool:**
- Return True only when ALL three hold:
  1. bool(envelope.get("title", "").strip())
  2. len(envelope.get("body", "")) >= 200
  3. bool(envelope.get("source_url", "").strip())
- Contract: {"title": "T", "body": "x"*199, "source_url": "http://x"} → False

**_fetch_feed(url: str) -> list:**
- result = feedparser.parse(url)
- if result.bozo: print(f"WARNING: feed parse error for {url}: {result.bozo_exception}", file=sys.stderr)
- return list(result.entries)  — always a list, never raises
- Contract: if feedparser.parse returns mock with .entries = [e1, e2], returns [e1, e2]

**_select_entry(entries: list, topics: list[str]) -> dict | None:**
- topics_lower = [t.lower() for t in topics]
- Pass 1: for entry in entries: combine title + summary into one lowercased string; if any(kw in text for kw in topics_lower): return entry
- Pass 2 (no match): return entries[0] if entries else None
- Contract: entry[1].title contains "technology" and topics=["technology"] → returns entry[1]; no match → returns entries[0]

**_try_entries(entries: list, topics: list[str], max_attempts: int = 3) -> dict | None:**
- Build ordered candidate list: first call _select_entry to get keyword-preferred entry, then remaining entries in order, deduplicated, capped to max_attempts
- Implementation: use a seen set to avoid duplicates; start with _select_entry result if non-None; append remaining entries in order
- For each candidate (up to max_attempts): raw = _extract_body(candidate); body = clean_html(raw); envelope = {"title": candidate.get("title", ""), "body": body, "source_url": candidate.get("link", "")}; if _validate(envelope): return envelope
- If no valid envelope found: return None
- Contract: entries=[short, short, valid_long] with max_attempts=3 → returns envelope with body len >= 200

**fetch_article(config: dict) -> dict:**
- primary_url = config["primary_url"]; fallback_url = config["fallback_url"]; topics = config["content_topics"]
- Try primary: entries = _fetch_feed(primary_url); result = _try_entries(entries, topics) if entries else None
- If result is not None: return result
- Try fallback: entries = _fetch_feed(fallback_url); result = _try_entries(entries, topics) if entries else None
- If result is not None: return result
- Both failed: print("ERROR: both primary and fallback feeds failed to return a valid article", file=sys.stderr); sys.exit(1)
- Contract: primary returns [] → falls back; both return [] → sys.exit(1); mock returns valid entry → returns envelope dict with keys {title, body, source_url}

**_load_config() -> dict:**
- config_path = Path("plan/config.json")
- with config_path.open() as f: return json.load(f)["content_feeds"]
- No error handling needed (missing config is a fatal programmer error, Python will raise naturally)

**main() -> None:**
- today = get_beijing_date()
- if content_path(today).exists(): print(f"[feed_article] {today} already exists, skipping.", file=sys.stderr); sys.exit(0)
- config = _load_config()
- envelope = fetch_article(config)
- print(json.dumps(envelope))  # stdout

**Module entry point (REQUIRED — last 2 lines of file):**
```python
if __name__ == "__main__":
    main()
```
