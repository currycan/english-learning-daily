import os
from unittest.mock import patch, MagicMock
import pytest
import scripts.ai_provider as ap


# ---------------------------------------------------------------------------
# call_gemini tests
# ---------------------------------------------------------------------------


def test_call_gemini_returns_text():
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="response text")
        result = ap.call_gemini("test prompt")
    assert result == "response text"


def test_call_gemini_raises_provider_error_on_failure():
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.side_effect = Exception("api down")
        with pytest.raises(ap.ProviderError):
            ap.call_gemini("prompt")


def test_call_gemini_uses_env_api_key(monkeypatch):
    monkeypatch.setenv("GEMINI_API_KEY", "test-key-123")
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="ok")
        ap.call_gemini("prompt")
    MockClient.assert_called_once_with(api_key="test-key-123")


def test_call_gemini_uses_default_model():
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="ok")
        ap.call_gemini("prompt")
    call_kwargs = mock_client.models.generate_content.call_args
    assert call_kwargs.kwargs.get("model") == "gemini-2.0-flash-lite"


def test_call_gemini_uses_custom_model():
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="ok")
        ap.call_gemini("prompt", model="gemini-2.0-flash")
    call_kwargs = mock_client.models.generate_content.call_args
    assert call_kwargs.kwargs.get("model") == "gemini-2.0-flash"


def test_call_gemini_prefers_env_over_kwarg(monkeypatch):
    monkeypatch.setenv("GEMINI_API_KEY", "env-key")
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="ok")
        ap.call_gemini("prompt", api_key="kwarg-key")
    MockClient.assert_called_once_with(api_key="env-key")


def test_call_gemini_logs_model_to_stderr(capsys):
    with patch("scripts.ai_provider.genai.Client") as MockClient:
        mock_client = MagicMock()
        MockClient.return_value = mock_client
        mock_client.models.generate_content.return_value = MagicMock(text="ok")
        ap.call_gemini("prompt")
    captured = capsys.readouterr()
    assert "INFO: Gemini model: gemini-2.0-flash-lite" in captured.err

# AI Provider Configuration / AI 提供商配置

> English . 中文对照

---

## Overview / 概述

This system uses Google Gemini as its content generation backend via the `google-genai` SDK. A single API key (`GEMINI_API_KEY`) is required. The default model is `gemini-2.0-flash-lite`.

本系统使用 Google Gemini 作为内容生成后端，通过 `google-genai` SDK 调用。需要一个 API 密钥 (`GEMINI_API_KEY`)。默认模型为 `gemini-2.0-flash-lite`。

---

## 1. Get a Gemini API Key / 获取 Gemini API Key

Follow these steps to obtain an API key from [aistudio.google.com](https://aistudio.google.com):

按以下步骤从 [aistudio.google.com](https://aistudio.google.com) 获取 API key：

1. Visit [aistudio.google.com](https://aistudio.google.com) and sign in with your Google account.

   访问 [aistudio.google.com](https://aistudio.google.com) 并使用 Google 账号登录。

2. Click **Get API key** in the left sidebar.

   在左侧边栏点击 **Get API key**。

3. Click **Create API key** and select a Google Cloud project (or create one).

   点击 **Create API key** 并选择 Google Cloud 项目（或新建）。

4. Copy the key immediately — store it securely (e.g., a password manager).

   立即复制密钥——请安全存储（如密码管理器）。

**Cost note:** `gemini-2.0-flash-lite` (~1,400 tokens/day) is within the free tier for most usage patterns.

**费用说明：** `gemini-2.0-flash-lite`（每天约 1,400 tokens）在大多数使用场景下处于免费额度内。

---

## 2. Add Key as GitHub Actions Secret / 添加为 GitHub Actions Secret

`GEMINI_API_KEY` must be added as a GitHub Repository Secret so the CI workflow can access it. The key is never stored in code or config files.

`GEMINI_API_KEY` 须添加为 GitHub 仓库 Secret，以便 CI 工作流访问。密钥不存储在代码或配置文件中。

In your GitHub repository:

在 GitHub 仓库中：

\```
Settings -> Secrets and variables -> Actions -> New repository secret
  Name:  GEMINI_API_KEY
  Value: <your Gemini API key>
\```

---

## 3. Configuration / 配置

### Default model / 默认模型

The default model is set in `plan/config.json`:

默认模型在 `plan/config.json` 中设置：

\```json
{
  "gemini_model": "gemini-2.0-flash-lite"
}
\```

Override by changing the `gemini_model` value. The code constant `GEMINI_MODEL` in `scripts/ai_provider.py` serves as a fallback if the config field is absent.

修改 `gemini_model` 值即可覆盖。`scripts/ai_provider.py` 中的代码常量 `GEMINI_MODEL` 在配置字段缺失时作为回退值。

### API key resolution priority / API 密钥解析优先级

1. `GEMINI_API_KEY` environment variable (highest priority).

   `GEMINI_API_KEY` 环境变量（最高优先级）。

2. `api_key` parameter passed to `call_gemini()` (code-level override).

   传递给 `call_gemini()` 的 `api_key` 参数（代码级覆盖）。

---

## 4. SDK Details / SDK 详情

| Property / 属性 | Value / 值 |
|---|---|
| Package | `google-genai` (install: `pip install google-genai`) |
| Import | `from google import genai` |
| Client | `genai.Client(api_key=key)` |
| Call | `client.models.generate_content(model=..., contents=prompt)` |
| Response | `response.text` |

---

## Summary / 配置总结

| Secret / 密钥 | Required By / 所需功能 |
|---|---|
| `GEMINI_API_KEY` | Gemini API calls via `call_gemini()` |
