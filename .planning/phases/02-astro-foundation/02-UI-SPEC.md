# Phase 02 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Scope note:** Phase 02 delivers infrastructure proof only — a minimal lesson list and stub lesson pages.
Full reading UX (typography, collapsibles, tap-to-reveal) belongs to Phase 03. This contract specifies only
what is built in Phase 02 and sets the token baseline that Phase 03 extends.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none — Tailwind 4 utility classes only |
| Preset | not applicable |
| Component library | none — native HTML + Astro components |
| Icon library | none — Phase 02 uses no icons |
| Font | system-ui (browser default stack — no Google Fonts or custom fonts in Phase 02) |

Source: RESEARCH.md Standard Stack — Tailwind 4 via `@tailwindcss/vite`; no component library declared.
shadcn not applicable — stack is Astro `.astro` files, not React.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element and list-item spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Touch target minimum 44px (REQUIREMENTS.md LESS-09 — applies from Phase 03 forward;
not enforced in Phase 02 stub pages but declared here as a baseline rule for all phases).

Source: REQUIREMENTS.md LESS-09 declares 44px touch target minimum.

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

Implementation: Tailwind 4 utility classes — `text-base` (16px), `text-sm` (14px), `text-xl` (20px),
`text-2xl` (28px). Font weights: `font-normal` (400), `font-semibold` (600).

Phase 02 pages use Body (16px/400) and Heading (20px/600) only. Label and Display are declared here
for Phase 03 continuity.

Source: REQUIREMENTS.md LESS-09 mandates ≥16px body text. All other values are defaults.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #ffffff (light) / #0f172a (dark) | Page background, content area |
| Secondary (30%) | #f8fafc (light) / #1e293b (dark) | List container, page wrapper |
| Accent (10%) | #2563eb | Lesson date links only |
| Destructive | #dc2626 | Not used in Phase 02 |

Accent reserved for: lesson date link text on the lesson list page only. No other elements use the
accent color in Phase 02.

Implementation note: Phase 02 delivers light mode only. Dark mode tokens are declared here for Phase 05
continuity but are not wired up until Phase 05 (CONTEXT.md D-09 defers FOUC/theme to Phase 05).

Source: CONTEXT.md confirms minimal styling. REQUIREMENTS.md PROG-03/PROG-06 assign dark mode to Phase 05.
Color values are defaults for a clean light-mode site.

## Component Inventory

Minimal set for Phase 02. Phase 03 extends this list.

| Component | File | Description |
|-----------|------|-------------|
| Base layout | `website/src/layouts/Base.astro` | HTML shell: `<head>` with viewport meta, title slot, Tailwind global import, `<slot />` body |
| Lesson list page | `website/src/pages/index.astro` | Reads all lessons via `getCollection`, sorts descending, renders `<ul>` of date links |
| Stub lesson page | `website/src/pages/lesson/[id].astro` | Dynamic route; renders `<Content />` from `render(entry)` — raw Markdown output |
| Global styles | `website/src/styles/global.css` | `@import "tailwindcss"` only — no custom tokens in Phase 02 |

No shared UI components beyond the base layout. Lesson list and stub pages are self-contained pages.

---

## Interaction Contracts

Phase 02 has no interactive behavior beyond standard anchor navigation.

| Interaction | Behavior | Implementation |
|-------------|----------|----------------|
| Lesson link click | Navigate to `/lesson/YYYY-MM-DD` | Native `<a>` — no JS required |
| Browser back | Return to lesson list | Native browser history — no JS required |
| 404 route | Astro default 404 page | No custom 404 page in Phase 02 |

No JavaScript shipped in Phase 02 pages. All interactivity is deferred to Phase 03+.

## Layout Contract

| Property | Value |
|----------|-------|
| Page max-width | 640px (centered with `mx-auto`) |
| Page padding (horizontal) | 16px (`px-4`) |
| Page padding (vertical) | 32px (`py-8`) |
| List item spacing | 8px (`space-y-2` on `<ul>`) |
| Base font size | 16px (`text-base` on `<body>`) |

The lesson list page renders a `<ul>` with one `<li>` per lesson. Each `<li>` contains a single
`<a>` with the date string as link text.
