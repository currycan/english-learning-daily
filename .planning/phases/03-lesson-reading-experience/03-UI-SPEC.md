## Spacing Scale

Declared values (multiples of 4 only). All enforced via Tailwind 4 utility classes.

| Token | Value | Tailwind Class | Usage |
|-------|-------|---------------|-------|
| xs | 4px | `gap-1`, `px-1` | Icon-to-text gaps, tight inline padding |
| sm | 8px | `gap-2`, `px-2`, `mb-2` | Compact element spacing within cards |
| md | 16px | `px-4`, `mb-4`, `py-4` | Default element spacing, section inner padding |
| lg | 24px | `px-6`, `mb-6`, `py-6` | Section header padding, card outer spacing |
| xl | 32px | `mb-8`, `py-8` | Between major page sections |
| 2xl | 48px | `py-12` | Top-of-page breathing room (homepage) |
| 3xl | 64px | — | Reserved for future phase (Phase 05 polish) |

Exceptions:
- **Tap target minimum: 44px** — All interactive elements (section collapse headers, answer reveal buttons, lesson list links) must have `min-h-[44px]` and sufficient horizontal padding. Source: REQUIREMENTS.md LESS-09, CONTEXT.md D-11.
- **Container width: max-w-[640px]** — Already established in `Base.astro`. Do not override. Source: CONTEXT.md code_context.

## Typography

All sizes in pixels. Two weights maximum. Enforced via Tailwind 4 utilities.

| Role | Size | Tailwind Size | Weight | Tailwind Weight | Line Height | Usage |
|------|------|---------------|--------|----------------|-------------|-------|
| Body | 16px | `text-base` | 400 regular | `font-normal` | 1.625 (`leading-relaxed`) | Article paragraphs, definition text, Q&A answers |
| Label | 14px | `text-sm` | 400 regular | `font-normal` | 1.25 (`leading-tight`) | Vocabulary POS tags, Chinese gloss, quote attribution, date stamps |
| Heading | 20px | `text-xl` | 600 semibold | `font-semibold` | 1.2 (`leading-tight`) | Section collapse headers (📚 词汇, 🔗 表达, ❓ 理解), page h1 |
| Display | 28px | `text-[28px]` | 600 semibold | `font-semibold` | 1.2 (`leading-tight`) | Homepage featured lesson title (lesson date as display heading) |

Notes:
- `leading-relaxed` is already set on `<body>` in `Base.astro` (`class="text-base leading-relaxed"`). Body inherits this.
- Only two weights used: 400 (`font-normal`) and 600 (`font-semibold`). No 700/bold.
- Display size `text-[28px]` uses Tailwind 4 arbitrary value — valid syntax.
- Source: REQUIREMENTS.md LESS-09 (≥16px body), website-CONTEXT.md D-24, Base.astro existing classes.

## Color

Tailwind CSS 4 with no config file. Colors expressed as arbitrary values or Tailwind default palette classes where available. Phase 05 will introduce full dark/light theming — this phase inherits system default only (no manual toggle yet).

| Role | Value | Tailwind Expression | Usage |
|------|-------|---------------------|-------|
| Dominant (60%) | `#ffffff` (light) / system default | `bg-white` or default body | Page background, article text area |
| Secondary (30%) | `#f9fafb` (gray-50) | `bg-gray-50` | Vocabulary definition cards, chunking expression cards, collapsed section placeholder |
| Accent (10%) | `#2563eb` (blue-600) | `text-[#2563eb]` or `text-blue-600` | Reserved for: lesson list links (homepage), "阅读原文 →" source link, "查看答案" reveal button text |
| Section border | `#93c5fd` (blue-300) | `border-blue-300` | Left border on vocabulary definition cards (`border-l-4`) |
| Notice bar | `#fef08a` (yellow-200) bg + `#854d0e` (yellow-900) text | `bg-yellow-200 text-yellow-900` | "⚠️ 今日课文将在中午更新" notice bar only |
| Muted text | `#6b7280` (gray-500) | `text-gray-500` | POS tags, Chinese gloss, quote text, date labels in recent list |
| Destructive | n/a | n/a | No destructive actions in this phase |

Accent (`#2563eb`) reserved for:
1. Homepage recent lesson list `<a>` links
2. "阅读原文 →" original source link (one per lesson)
3. "查看答案" reveal button text label

Accent is NOT used for: section collapse headers, card backgrounds, borders, or any decorative purpose.

Source: `#2563eb` established in Phase 02 `index.astro` stub (`class="text-[#2563eb]"`). Carries forward.

## Interaction Contract

### Collapsible Sections (LESS-05, LESS-06, LESS-07)

| Property | Value |
|----------|-------|
| Default state | Collapsed (`data-collapsed="true"`) |
| Trigger | Tap/click anywhere on section header button (full width) |
| Animation | CSS `grid-template-rows: 0fr → 1fr` transition |
| Duration | 300ms |
| Easing | `ease-out` |
| Chevron | `▼` rotates to `▲` (180deg rotate, `transition-transform duration-200`) |
| Implementation | `<script is:inline>` placed at end of `<body>`, after all collapsible elements |
| Tap target | Section header button: `min-h-[44px] w-full text-left py-3 px-0` |
| Sections | 📚 词汇 / Vocabulary, 🔗 表达 / Chunking Expressions, ❓ 理解 / Comprehension |

**No native `<details>/<summary>`** — locked out by CONTEXT.md D-07.

Fallback if `grid-template-rows` transition proves problematic: `max-height: 0 → 2000px`, `transition: max-height 0.3s ease-out`. Use only if grid approach fails at implementation time.

### Tap-to-Reveal Answers (LESS-08)

| Property | Value |
|----------|-------|
| Default state | Answer block hidden (`class="hidden"`) |
| Trigger | Tap "查看答案" button for each Q&A pair |
| Reveal behavior | Remove `hidden` class from answer block; remove or hide the "查看答案" button (one-way reveal, no re-hide) |
| Reveals together | English answer (`A (EN)`) and Chinese answer (`A (中)`) shown in the same block, both revealed in one tap |
| Tap target | "查看答案" button: `min-h-[44px] px-2` |
| Button style | `text-sm text-blue-600 underline` |

### Notice Bar (LESS-03)

| Property | Value |
|----------|-------|
| Trigger | Build-time conditional: shown when no lesson with today's Asia/Shanghai date exists |
| Position | Top of page, above the featured lesson, inside the `Base.astro` `<main>` slot |
| Style | `bg-yellow-200 text-yellow-900 px-4 py-3 rounded text-sm font-normal mb-4` |
| Copy | `⚠️ 今日课文将在中午更新` |
| Behavior | Static — no dismiss, no JS required |

### Source Link (D-02)

| Property | Value |
|----------|-------|
| Label | `阅读原文 →` |
| Style | `text-blue-600 text-sm underline` |
| Position | End of article section, before the collapsible sections begin |
| Extraction | Regex on article section: `/^>\s*Source:\s*(https?:\/\/\S+)/m` |

### localStorage Write (PROG-01 — Phase 04 dependency)

| Property | Value |
|----------|-------|
| Trigger | Auto-fires when any lesson page (`/lesson/[id]`) is loaded |
| Implementation | `<script is:inline>` in `[id].astro` |
| Key | `read_lessons` |
| Value format | JSON array of ISO date strings: `["2026-03-24", "2026-03-23"]` |
| Behavior | Read existing array from localStorage → add current `lesson.id` if not present → write back (immutable: always create a new array, never push to existing reference) |
