# Claude Code 项目配置

本目录包含针对 English Learning Daily 项目的 Claude Code 专用配置。

## 文件说明

### settings.local.json
项目特定的权限配置，按功能分组：

- **核心功能**：Python脚本执行、学习进度管理
- **版本控制**：Git基础操作
- **开发工具**：测试、包管理、虚拟环境
- **GitHub集成**：仓库操作、认证
- **Web资源**：内容获取、搜索

### agents/
项目专用的 Claude 代理：

- **progress.md**：查看学习进度和统计
- **fix-workflow.md**：诊断GitHub Actions故障
- **preview.md**：预览推送通知内容

## 设计原则

1. **最小权限**：只授权项目必需功能
2. **功能分组**：按用途组织权限，便于管理
3. **项目专用**：针对英语学习自动化的特定需求

## 使用方式

这些配置会自动生效，提供：
- 无需重复授权的开发体验
- 项目专用的调试和管理工具
- 团队共享的操作流程
